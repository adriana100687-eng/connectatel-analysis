# 📊 ConnectaTel Customer Analysis

## 🎯 Objective
Analyze customer behavior for ConnectaTel to identify usage patterns, detect data quality issues, and segment customers based on their usage and age.

---

## 📂 Datasets
- plans.csv → plan details (pricing, minutes, data, extra costs)  
- users_latam.csv → customer information (age, city, registration date, plan, churn)  
- usage.csv → usage behavior (calls and messages)  

---

## 🧩 Analysis Steps
1. Data loading and exploration  
2. Data quality checks (missing values, invalid values, dates)  
3. Data cleaning (sentinels, nulls, formatting)  
4. Aggregation of usage metrics per user  
5. Visualization and outlier detection  
6. Customer segmentation (usage and age)  

---

## ▶️ How to Run
Open the notebook using:
- Jupyter Notebook  
- Google Colab  

Run all cells from top to bottom.

---

## 🔁 Reproducibility Guide
1. Load the datasets from `/datasets/`  
2. Install required libraries: pandas, numpy, seaborn, matplotlib  
3. Execute all cells in order
